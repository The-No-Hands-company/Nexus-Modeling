# v0.20 Planning — Mueller Matrix BSDF, Time-Resolved Phosphorescence, Hyperspectral IBL

Status: **Planning**. Three passes completing the advanced polarisation and spectral
lighting stack deferred from v0.19.

Reference: [ROADMAP.md](../ROADMAP.md), [v0.19-planning.md](v0.19-planning.md).

---

## Motivation

v0.19 delivered polarisation rendering, fluorescence simulation, and spectral upsampling
from RGB assets. Three items were explicitly deferred:

1. **Mueller Matrix BSDF Representation** — extends polarisation rendering with full 4×4
   Mueller matrix evaluation at each scattering event; required for accurate polarised
   BRDF/BTDF on anisotropic metals, coated surfaces, and birefringent media.
2. **Time-Resolved Phosphorescence Decay** — accumulates phosphorescence emission over
   multiple frames with configurable exponential decay; enables glow-in-the-dark and
   afterglow effects using the phosphorescence simulation infrastructure from v0.19.
3. **Hyperspectral Image-Based Lighting** — extends IBL with a full spectral radiance
   environment map (sampled per wavelength band); feeds into multi-spectral path tracing
   passes for physically-accurate sky and environment illumination across the visible
   spectrum.

## Out of Scope for v0.20

- Plenoptic / light-field rendering (deferred to v0.21).
- Coherent wave optics (diffraction, interference — deferred to v0.21).
- Spectral participating media with wavelength-dependent extinction (deferred to v0.21).

## Exit Criteria

1. `enableMuellerBSDF` triggers Mueller matrix dispatch; `muellerBSDFActive` and
   `muellerBSDFEvalCount` populated.
2. `enablePhosphorescenceDecay` triggers decay accumulation; `phosphorescenceDecayActive`
   and `phosphorescenceDecayFrames` populated.
3. `enableHyperspectralIBL` triggers spectral env dispatch; `hyperspectralIBLActive`
   and `hyperspectralIBLBandCount` populated.
4. All flags default to off — no behavior change for existing callers.
5. All new Null-backend tests pass (target: +24 tests).
6. Null-backend full suite at 100%.
