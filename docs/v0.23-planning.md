# v0.23 Planning — Instant-NGP Hash-Grid NeRF, Dynamic NeRF, Holographic Wavefront Encoding

Status: **Planning**. Three passes completing the neural rendering and display stack
deferred from v0.22.

Reference: [ROADMAP.md](../ROADMAP.md), [v0.22-planning.md](v0.22-planning.md).

---

## Motivation

v0.22 delivered NeRF ray marching, Gaussian splatting spectral extensions, and
light-field display output. Three items were explicitly deferred:

1. **Instant-NGP Hash-Grid Acceleration for NeRF** — replaces the dense MLP positional
   encoding with a multi-resolution hash grid, enabling an order-of-magnitude speedup in
   NeRF inference via smaller network width and cached spatial features.
2. **Dynamic NeRF (D-NeRF, time-conditioned)** — extends the base NeRF radiance field
   with a time warp network; enables rendering of deformable/animated scenes captured
   from multi-view video.
3. **Holographic Wavefront Encoding** — generates per-depth-slice complex wavefront
   tiles for holographic display targets; encodes amplitude + phase via a pupil-plane
   propagation pass.

## Exit Criteria

1. `enableInstantNGP` triggers hash-grid dispatch; `instantNGPActive` and
   `instantNGPHashLevels` populated.

   `holographicWavefrontActive` and `holographicWavefrontSliceCount` populated.
4. All flags default to off — no behavior change for existing callers.
5. All new Null-backend tests pass (target: +24 tests).
6. Null-backend full suite at 100%.
