# v0.24 Planning — Neural Scene Flow, Eyebox Steering, NeRF-in-the-Wild

Status: **Planning**. Three passes completing the advanced neural and holographic
rendering stack deferred from v0.23.

Reference: [ROADMAP.md](../ROADMAP.md), [v0.23-planning.md](v0.23-planning.md).

---

## Motivation

v0.23 delivered Instant-NGP hash-grid NeRF acceleration, Dynamic NeRF (D-NeRF), and
holographic wavefront encoding. Three items were explicitly deferred:

1. **Neural Scene Flow / 4D Occupancy Grids** — estimates per-point 3D motion vectors
   from dynamic scene volumes; used for temporally consistent novel-view synthesis and
   4D occupancy reconstruction from multi-view video.
2. **Eyebox / Exit-Pupil Steering for Holographic Near-Eye Displays** — steers the
   holographic exit pupil to track the user's gaze; generates eye-position-dependent
   wavefront corrections for near-eye holographic displays.
3. **NeRF-in-the-Wild (Appearance Embedding, Transient Distractors)** — extends NeRF
   with per-image appearance latent codes and a transient-object prediction head;
   enables robust reconstruction from in-the-wild internet photo collections.

## Exit Criteria

1. `enableNeuralSceneFlow` triggers flow+occupancy dispatch; `neuralSceneFlowActive` and
   `neuralSceneFlowVoxelCount` populated.
2. `enableEyeboxSteering` triggers pupil-correction dispatch; `eyeboxSteeringActive` and
   `eyeboxSteeringSliceCount` populated.
3. `enableNeRFWild` triggers appearance+transient dispatch; `neRFWildActive` and
   `neRFWildTransientPasses` populated.
4. All flags default to off — no behavior change for existing callers.
5. All new Null-backend tests pass (target: +24 tests).
6. Null-backend full suite at 100%.
